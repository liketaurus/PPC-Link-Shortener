About PPC Link Shortener

Just a little link shortener 'service' created as a 'replacement' of goo.gl, intended for a 'local' use only.

Project was created using the following:

  •  Bootstrap 4
  •  Bare Bootstrap template
  •  jQuery
  •  jQuery QRcode
  •  Dexie.js
  •  Firebase Dynamic Links REST API
  •  Font Awesome
  •  clipboard.js
